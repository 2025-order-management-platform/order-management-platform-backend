
# <음식 주문 관리 플랫폼>
## 프로젝트 목적 / 상세 
- 이 플랫폼은 온라인과 오프라인 모두에서 고객이 움식을 주문하고 결제할 수 있도록 지원합니다.
- 서비스는 음식점 주인, 고객, 관리자 등 세 가지 주요 사용자에 초점을 맞춰 각 사용자에게 맞는 권한과 기능을 제공합니다.
- ai 기능을 통해 음식 설명을 자동으로 생성을 통해 고객과 주인 모두의 편의를 도모함.
- 다각적인 상품 검색 기능을 통한 고객 만족도 향상을 도모함(?)
